# Dive Into Deep Learning


## Versions
miniconda: Miniconda3-py39_25.9.1-1-Linux-x86_64.sh

## Instructions

1. Download miniconda.
2. Set up python interpreter in pycharm to target a new conda environment named d2l that uses python 3.9.
3. Initialize conda shell: `$ ~/miniconda3/bin/conda init`.
4. Close and re-open shell.
5. `$ conda activate d2l`
6. Install CUDA (if applicable).
7. `$ pip install torch==2.0.0 torchvision==0.15.1`
8. `$ pip install d2l==1.0.3`
9. Go to D2l and click notebooks at top of any page to download notebooks.
10. `$ unzip d2l-en.zip && rm d2l-en.zip`
11. `$ jupyter notebook`
